# 100ms Integration

Live Agora uses [100ms](https://www.100ms.live/) for video-calling functionality (live video/audio streams). For instance administrators, the following one-time setup is required.

## 1. Create a 100ms account and setup credentials

1. Create an account at https://www.100ms.live/
2. (Optional) Create a workspace
3. Find your "App Access Key" and "App Secret" in the Dashboard > Developer page (https://dashboard.100ms.live/developer)
4. Add values to `packages/server/.env` file:
    ```
    ...
    100MS_APP_ACCESS_KEY=your_100ms_app_access_key_here
    100MS_APP_SECRET=your_100ms_app_secret_here
    ...
    ```

## 2. Generate the room template

1. Generate the template and upload to 100ms

    ```bash
    cd packages/server
    node --env-file=.env ./helper-scripts/generateHmsTemplate3.cjs
    ```

2. In 100ms dashboard, set the template as default
   1. Go to Dashboard > Templates > default_liveagora_template
   2. Tap the three dots and tap "Make Default"
